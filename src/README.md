У даному репозиторії представлено результати виконання циклу практичних та лабораторних робіт з фронтенд-розробки проєкту «Галерея Сучасного Мистецтва». 

Практична робота №1
Тема: Вступ до React. Управління станом і подіями.
Мета: Опанування компонентного підходу React, передача даних через пропси, керування локальним станом та реалізація взаємодії з користувачем через форми.
Завдання: Створено каркас головної сторінки додатка, що відображає список об'єктів та дозволяє здійснювати їх первинну фільтрацію.

Практична робота №2
Тема: Маршрутизація та управління формами в React за допомогою Zod.
Мета: Створення багатосторінкового додатка за допомогою React Router та надійних форм із декларативною валідацією.
Завдання: Реалізовано систему маршрутів (/, /register/:eventId, /participants/:eventId) та навігацію за допомогою Link та useNavigate.

Лабораторна робота №2
Тема: Глобальне управління станом за допомогою Redux Toolkit.
Мета: Централізоване керування станом додатка та синхронізація даних через Redux Thunk.
Завдання: Логіку збереження подій та учасників перенесено у глобальне сховище (Store). Реалізовано асинхронні запити для завантаження учасників та динамічну фільтрацію на рівні Redux-селекторів.

Лабораторна робота №3
Тема: Агрегація даних та візуалізація аналітики.
Мета: Візуалізація складних даних та підготовка проєкту до деплою.
Завдання: Створено сторінку аналітики з інтерактивними графіками (Recharts). Реалізовано адаптивний дизайн для мобільних пристроїв та розгорнуто проєкт на хостингу Vercel.

Інструкція з запуску:

Виконати команду npm install для встановлення залежностей.

Запустити локальний сервер командою npm start.
